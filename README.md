This is a simple third-person fighting game made with Unity.  
It is a small prototype created to test basic combat and gameplay mechanics.

The game does not have a story or objectives. It is focused only on gameplay testing.

## Controls

- Move: **WASD**
- Pick up item: **E**
- Drop item: **Q**
- Attack: **Left Mouse Button**
- Block: **Right Mouse Button**
- Change inventory slot: **1, 2, 3, 4**

## Features

- Third-person movement
- Simple combat system
- Three weapon types with different range and damage
- Basic inventory system
- Health potions
- Enemy AI that follows and attacks the player
- Visual elements like health bars and pickup indicators

## Requirements

- Unity 2020.3.32
- C#
